1076:求x的n次幂

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入 1 个正实数x 和1 个正整数 n(n<=10)，计算并输出x 的n 次幂（保留2 位小数）。

Sample Input

2
1.5 2
2.0 10

Sample Output

2.25
1024.00

#include<stdio.h>
main()
{
double a,b;
int repeat,n,i,l;
scanf("%d",&repeat);
if(repeat<10&&repeat>0)
    {	
    for(l=0;l<repeat;l++)
        {			
        scanf("%lf %d",&a,&n);
            if(n<=10)
            {
            b=a;
            for(i=1;i<n;i++)
                a=a*b;
            printf("%.2lf\n",a);
            }
        }
    }	
}

1073:字母转换

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入一个字符，如果它是大写字母，输出相应的小写字母；如果它是小写字母，输出相应的
大写字母；否则，原样输出。

Sample Input

3F=y

Sample Output

f=Y 

#include<stdio.h>
main()
{
char a[3];
int repeat,i,l;
scanf("%d",&repeat);
    for(l=0;l<repeat;l++)
    {	
        scanf("%s",&a);
        for(i=0;i<=3;i++)
        {
        if(a[i]>='A'&&a[i]<='Z')
            a[i]=a[i]+32;
        else if(a[i]>='a'&&a[i]<='z')
            a[i]=a[i]-32;
        else 
            a[i]=a[i];
        }
        printf("%s",a);
    }
}

1072:显示水果的价格

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
以下 4 种水果的单价分别是3.00 元/公斤，2.50 元/公斤，4.10 元/公斤，10.20 元/公斤。
[1] apples
[2] pears
[3] oranges
[4] grapes
输入水果的编号，输出该水果的单价(保留2 位小数)。如果输入不正确的编号，显示单价为0.00。

Sample Input

1
3

Sample Output

price=4.10

#include<stdio.h>
main()
{
int a,b,i;
double price;
scanf("%d",&b);
for(i=0;i<b;i++)
    {
    scanf("%d",&a);
    switch(a)
        {
        case 1:printf("price=3.00\n");break;
        case 2:printf("price=2.50\n");break;
        case 3:printf("price=4.10\n");break;
        case 4:printf("price=10.20\n");break;
        default:printf("price=0.00\n");break;
        }
    }
}

1071:计算个人所得税

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入一个职工的月薪 salary，输出应交的个人所得税tax(保留2 位小数)。
tax = rate * (salary-850)
当 salary <= 850 时，rate = 0;
当 850 < salary <= 1350 时，rate = 5;
当 1350 < salary <= 2850 时，rate = 10;
当 2850 < salary <= 5850 时，rate = 15;
当 5850 < salary 时，rate = 20;

Sample Input

2
200
2000
Sample Output
tax=0.00
tax=115.00

HINT

当结果为0时，你的程序可能输出的是 -0.00，请判断为0时，使用printf(“tax=0.00\n”);

#include<stdio.h>
main()
{
double tax;
int b,rate,salary,i;
scanf("%d",&b);
    for(i=0;i<b;i++)
    {	
        scanf("%d",&salary);
        if(salary>5850)
        {
            rate=20;
        }	
        else if(salary>2850)
        {
            rate = 15;
        }
        else if(salary>1350)
        {
            rate=10;
        }	
        else if(salary>850)
        {
            rate=5;	
        }
        else
        {
            rate=0;
        }
        tax=rate*(salary-850)*0.01;
        if(tax==0)
        {
            printf("tax=0.00\n");
        }		
        else
        {
            printf("tax=%.2lf\n",tax);
        }
    }	
        return 0;
}
