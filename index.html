1072:显示水果的价格

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
以下 4 种水果的单价分别是3.00 元/公斤，2.50 元/公斤，4.10 元/公斤，10.20 元/公斤。
[1] apples
[2] pears
[3] oranges
[4] grapes
输入水果的编号，输出该水果的单价(保留2 位小数)。如果输入不正确的编号，显示单价为0.00。

Sample Input

1
3

Sample Output

price=4.10

#代码
#include<stdio.h>
main()
{
int a,b,i;
double price;
scanf(“%d”,&b);
for(i=0;i<b;i++)
{
scanf(“%d”,&a);
switch(a)
{
case 1:printf(“price=3.00\n”);break;
case 2:printf(“price=2.50\n”);break;
case 3:printf(“price=4.10\n”);break;
case 4:printf(“price=10.20\n”);break;
default:printf(“price=0.00\n”);break;
}
}
}
用上switch case函数了，属实人上人

1071:计算个人所得税

#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入一个职工的月薪 salary，输出应交的个人所得税tax(保留2 位小数)。
tax = rate * (salary-850)
当 salary <= 850 时，rate = 0;
当 850 < salary <= 1350 时，rate = 5;
当 1350 < salary <= 2850 时，rate = 10;
当 2850 < salary <= 5850 时，rate = 15;
当 5850 < salary 时，rate = 20;

Sample Input

2
200
2000
Sample Output
tax=0.00
tax=115.00

HINT

当结果为0时，你的程序可能输出的是 -0.00，请判断为0时，使用printf(“tax=0.00\n”);

#代码
#include<stdio.h>
main()
{
double tax;
int b,rate,salary,i;
scanf(“%d”,&b);
for(i=0;i<b;i++)
{
scanf(“%d”,&salary);
if(salary>5850)
{
rate=20;
}
else if(salary>2850)
{
rate = 15;
}
else if(salary>1350)
{
rate=10;
}
else if(salary>850)
{
rate=5;
}
else
{
rate=0;
}
tax=rate*(salary-850)*0.01;
if(tax==0)
{
printf(“tax=0.00\n”);
}
else
{
printf(“tax=%.2lf\n”,tax);
}
}
return 0;
}
用了好多个else if，感觉最容易忘记的地方就是else if实际上已经包含了上面条件的反向条件，经常忘掉

1064:计算旅途时间 

Description
输入 2 个整数time1 和time2，表示火车的出发时间和到达时间，计算并输出旅途时间。
有效的时间范围是 0000 到2359，不需要考虑出发时间晚于到达时间的情况。

Input
输入只有两个整数

Output
求出火车的旅途时间

Sample Input
712 1411
Sample Output
The train journey time is 6 hrs 59 mins.

代码如下

#include<stdio.h>
main()
{
    int time1,time2;
    int h,j,k,l,sj1,sj2,sj3,sj4;
    scanf("%d",&time1);
    scanf("%d",&time2);
    h=time1/100;
    j=time2/100;
    k=time1%100;
    l=time2%100;
    sj2=j*60+l;
    sj1=h*60+k;
    sj3=(sj2-sj1)/60;
    sj4=(sj2-sj1)%60;
    printf("The train journey time is %d hrs %d mins.",sj3,sj4);
}
