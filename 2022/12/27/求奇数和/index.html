#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入一批正整数(以零或负数为结束标志)，求其中的奇数和。

Sample Input

2
1 3 90 7 0
8 7 4 3 70 5 6 101 -1

Sample Output

11
116

#代码
#include <stdio.h>
int main( )
{
int a=1, sum=0,repeat,l;
scanf(“%d”, &repeat) ;
if(repeat<10&&repeat>0)
{
for(l=1;l<=repeat;l++)
{
while (a>0)
{
scanf(“%d”, &a) ;
if(a<=0)
{
a=1;
break;
}
else if(a%2!=0)
sum=sum+a ;
}
printf(“%d\n”,sum);
sum=0;

        }
    }
    return 0 ;	
}
