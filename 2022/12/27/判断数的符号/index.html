#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入整数 x，若x 大于0，y=1；若x 等于0，y=0；否则，y=-1，最后输出y。

Input

见sample

Output

见sample

Sample Input

3
2 -8 0

Sample Output

1
-1
0

#include<stdio.h>
main()
{
    int x,y,repeat,i;
    scanf("%d",&repeat);
    for(i=0;i<repeat;i++)
        {
        scanf("%d",&x);	
        if(x>0)
            y=1;
        else if(x<0)
            y=-1;
        else
            y=0;
        printf("%d\n",y);
        }
}
