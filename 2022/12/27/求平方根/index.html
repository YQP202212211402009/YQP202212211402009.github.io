求平方根 

Description
输入 1 个实数x，计算并输出其平方根（保留1 位小数）

Input
输入一个实数x

Output
输出平方根

Sample Input
17
Sample Output
The square root of 17.0 is 4.1

代码如下

#include<stdio.h>
#include<math.h>
int main()
{
    double a,b;
    scanf("%lf",&a);
    printf("The square root of %.1lf is ",a);
    b=sqrt(a);
    printf("%.1lf",b);
}
