#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入 1 个正实数x 和1 个正整数 n(n<=10)，计算并输出x 的n 次幂（保留2 位小数）。

Sample Input

2
1.5 2
2.0 10

Sample Output

2.25
1024.00

#include<stdio.h>
main()
{
double a,b;
int repeat,n,i,l;
scanf("%d",&repeat);
if(repeat<10&&repeat>0)
    {	
    for(l=0;l<repeat;l++)
        {			
        scanf("%lf %d",&a,&n);
            if(n<=10)
            {
            b=a;
            for(i=1;i<n;i++)
                a=a*b;
            printf("%.2lf\n",a);
            }
        }
    }	
}
