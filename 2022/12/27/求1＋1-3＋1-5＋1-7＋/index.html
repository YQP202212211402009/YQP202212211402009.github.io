#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入 1 个正整数 n(n<=100)，计算并输出1＋1/3＋1/5＋1/7＋⋯⋯的前n 项和(保留3 位小数)。

Sample Input

1
3

Sample Output

1.533

#include<stdio.h>
main()
{
double sum=0;
int i=1,n,a,l,x;
scanf("%d",&a);
for(l=1;l<=a;l++)
{
    scanf("%d",&n);
    if(n<=100)
    {   
        while(x<n)
        {
        sum=sum+1.0/i;
        i=i+2;  
        x=x+1;
        } 
    }
    printf("%.3lf\n",sum);
    sum=0;
    i=1;
    x=0;
}
return 0;
}
