#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入 1 个正实数eps，计算并输出1－1/3＋1/5－1/7＋⋯⋯，直到最后一项的绝对值小于eps(保留6 位小数，不包括最后一项)。

Sample Input

1
1E-4

Sample Output

0.785348

#代码
#include<stdio.h>
int main(){
int n,x,c,f;
int a;
int i=1;
scanf(“%d”,&a);
for(i=1;i<=a;i++)
{
scanf(“%d %d”,&n,&x);
printf(“Celsius Fahrenheit\n”);
for(c=n;c<=x;c++)
{
f=32+c*9/5;
printf(“%d%10d\n”,c,f);
}
}
return 0;
}
