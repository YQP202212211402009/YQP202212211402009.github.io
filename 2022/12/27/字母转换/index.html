#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入一个字符，如果它是大写字母，输出相应的小写字母；如果它是小写字母，输出相应的
大写字母；否则，原样输出。

Sample Input

3F=y

Sample Output

f=Y

#代码
#include<stdio.h>
main()
{
char a[3];
int repeat,i,l;
scanf(“%d”,&repeat);
for(l=0;l<repeat;l++)
{
scanf(“%s”,&a);
for(i=0;i<=3;i++)
{
if(a[i]>=’A’&&a[i]<=’Z’)
a[i]=a[i]+32;
else if(a[i]>=’a’&&a[i]<=’z’)
a[i]=a[i]-32;
else
a[i]=a[i];
}
printf(“%s”,a);
}
}
