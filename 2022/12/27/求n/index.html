Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
读入 1 个正整数 n (n<=22)，计算并输出 n! 。

Sample Input

3
1
10
5

Sample Output

1
3628800
120

{

for(l=0;l<a;l++)
{
scanf(“%d”,&n);
if(n<=22)
{
for(i=1;i<=n;i++)
sum=sum*i;

}
}
}
