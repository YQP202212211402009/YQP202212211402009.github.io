#题目
Description

输入一个正整数 repeat (0<repeat<10)，做repeat 次下列运算：
输入一个职工的月薪 salary，输出应交的个人所得税tax(保留2 位小数)。
tax = rate * (salary-850)
当 salary <= 850 时，rate = 0;
当 850 < salary <= 1350 时，rate = 5;
当 1350 < salary <= 2850 时，rate = 10;
当 2850 < salary <= 5850 时，rate = 15;
当 5850 < salary 时，rate = 20;

Sample Input

2
200
2000
Sample Output
tax=0.00
tax=115.00

HINT

当结果为0时，你的程序可能输出的是 -0.00，请判断为0时，使用printf(“tax=0.00\n”);

#代码
#include<stdio.h>
main()
{
double tax;
int b,rate,salary,i;
scanf(“%d”,&b);
for(i=0;i<b;i++)
{
scanf(“%d”,&salary);
if(salary>5850)
{
rate=20;
}
else if(salary>2850)
{
rate = 15;
}
else if(salary>1350)
{
rate=10;
}
else if(salary>850)
{
rate=5;
}
else
{
rate=0;
}
tax=rate*(salary-850)*0.01;
if(tax==0)
{
printf(“tax=0.00\n”);
}
else
{
printf(“tax=%.2lf\n”,tax);
}
}
return 0;
}
