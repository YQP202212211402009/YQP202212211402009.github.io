求余

#题目
Description

小学题目，给定除数和被除数，求余数

Input

多组测试数据,每组测试数据包含两个整数n,k(1<=n<=10^2000,1<=k<=1000)

Output
?对于每组测试数据,输出n%k

Sample Input

12 4
12 5

Sample Output

0
2

说是小学题目 哥们没写出来

#include<stdio.h>
#include<string.h>
main()
{
    int sum=0;
    char a[2001];
    int k,i;
    scanf("%s %d",&a,&k);
    sum=a[0]-'0';
    for(i=1;i<strlen(a);i++)
    {
        if(sum<k)
        {
            sum=sum*10+(a[i]-'0');
        }
    }
}
